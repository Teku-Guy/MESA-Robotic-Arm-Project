# Coding Agent Tasks

## File Creation
- Create new source files in `src/` directory
- Create header files in `include/` directory
- Follow naming conventions: `snake_case.cpp` for source, `PascalCase.h` for headers

## Code Implementation
- Implement functions using `camelCase` naming
- Use explicit types (`int`, `uint8_t`, `bool`)
- Add blank lines between logical sections
- Keep line length under 100 characters

## Error Handling
- Check return values from critical operations
- Log errors with `Serial.println()`
- Handle edge cases in public APIs

## Code Review
- Verify 2-space indentation
- Check K&R brace style
- Ensure proper include organization
